## Getting Started

Follow these steps to set up the environment and run the package.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/han16/Deep-MOTIFs.git
   cd Deep-MOTIFs
   ```
2. **Create and activate a virtual environment:**  
   Windows
   ```bash
   python -m venv deepmotifs
   .\deepmotifs\Scripts\activate 
   ```
   macOS/Linux
   ```bash
   python3 -m venv deepmotifs
   source deepmotifs/bin/activate
   ```
   
2. **Install packages:**
   ```bash
   pip install –r requirements.txt 
   ```
   
3. **Run Deep MOTIFs:**
   ```bash
   python –m deep_motifs --project-root C:\your\path\to\Deep-MOTIFs 
   ```
